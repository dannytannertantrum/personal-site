Project: Groceries

Status: Done - use it here or view it on GitHub

Stack/tools: Entirely frontend; bootstrapped with Create-React-App and uses local storage

Description: I was happy enough using the ColorNote app on my phone for keeping a grocery list, but I wanted to give it my own flavor. I frequent Wegmans and Costco the most, so this allows me to filter by store (or alphabetically), and gives me that nerdy, terminal-style dark/bright-green theme with some hot magenta thrown in. If you enjoy it, by all means use it! Just remember, it uses local storage. If you clear history on your device, make sure groceries.jaimefarnan.com is on your list of sites to exempt from being cleared.
